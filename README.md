## 禮尚往來系統頁面構想
* **首頁:**
大面漂亮背景 填滿
buttom*2 送禮者 收禮者

* **送禮者頁面1:**
送禮者輸入對方的基本資料
下拉式清單*1 選預算ex. 200、500、700、1000 以此類推
text輸入槽*4 收禮者名稱 收禮者信箱 收禮者聯絡方式 收禮者住址
bottom*1 下一步

* **送禮者頁面2:**
設計卡片的介面(讓送禮者可以選擇跟設計禮物內的卡片)
下拉式清單*1 選擇卡片模板
小畫家功能*1 繪製卡片
buttom*3 清除 確定送出 略過

* **送禮者頁面3:**
付款介面
這裡我懶得打了 相信你一定有個概念

* **收禮者頁面1:**
確認收禮者的身分
text輸入槽*3   姓名 手機 E-mail驗證碼(通知信內包含的驗證碼)
buttom*1 下一步

* **收禮者頁面2:**
選擇禮物
img圖檔*10 放出10種不同商品的圖
buttom*2 其他選項(重新跑出10種不同的圖) 確認

* **收禮者頁面3:**
確認地址(default = 送禮者輸入內容)
text輸入槽*4 收禮者名稱 收禮者信箱 收禮者聯絡方式 收禮者住址
buttom*1 確認送出
